# dlux-calc-web

Laravel 12 project dengan Tailwind CSS dan Boxicons.

## Tech Stack

- Laravel 12
- Tailwind CSS v4
- Boxicons
- Vite

## Installation

```bash
composer install
npm install
```

## Development

```bash
npm run dev
```
